# Airbnb Pricing Analysis

## Overview
End-to-end data analytics project analysing Airbnb listing prices 
using Python. Built as a portfolio project combining economics 
intuition with data science methodology.

## Skills Demonstrated
- Exploratory Data Analysis (pandas, matplotlib)
- Feature Engineering (datetime extraction, encoding, binarisation)
- Machine Learning (Decision Tree, Random Forest via scikit-learn)
- Model Evaluation (MAE, RMSE, R-squared)
- Customer Segmentation (RFM Analysis)
- Uncertainty Quantification (Bootstrap confidence intervals)

## Project Structure
- `01_generate_dummy_data.ipynb` — Full analysis notebook
- `data/` — Generated datasets and output charts
## Key Findings
- Location and review score are the dominant price drivers
- Random Forest outperforms single Decision Tree on all metrics
- Entire home listings command a significant price premium

## Tools
Python, pandas, numpy, scikit-learn, matplotlib, scipy
